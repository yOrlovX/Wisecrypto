My pet app - Wisecrypto for cryptocurrency statistics.

IOS version:   15.2

Environments : MVVM Architecture,
               SwiftUI,
               Combine,
               CoreData,
               Async/await(some case of parsing JSON)

SPM :          KeychainAccess,
               SwiftUI CachedAsyncImage 

API:          https://www.coingecko.com/
